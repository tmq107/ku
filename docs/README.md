# kli docs

A keyboard-driven Kubernetes TUI. These pages cover how to run it, the keys,
and what each feature does.

- [Getting started](getting-started.md) - install, run, flags, themes.
- [Keybindings](keybindings.md) - every key, by context.
- [Features](features.md) - tables, logs, edit, shell, scale, delete, switching.

kli uses your default kubeconfig (`$KUBECONFIG`, then `~/.kube/config`) and the
current context unless you pass `--context`.
